# Crime Analysis Dashboard (Tableau)

## Overview
This project focuses on analyzing crime data using Tableau to identify patterns, trends, and key insights that can support data-driven decision-making. The goal was to transform raw data into interactive dashboards for better visualization and understanding of crime behavior.

## Data Preparation
- Imported dataset into Tableau Public  
- Verified and corrected data types (Date, Year, Arrest, Domestic)  
- Created calculated fields:
  - Crime Hour  
  - Day of Week  
  - Time Block (Morning, Afternoon, Evening, Night)  
  - Severity Level  
- Mapped geographic data using Latitude and Longitude  

## Dashboards Created

### 1. Overall Crime Statistics
- Total crime KPI  
- Most common crime type  
- Geo-map showing crime locations  
- Recent crime activity table  

### 2. Time-Based Analysis
- Crime distribution by day of week  
- Crime trend by hour of day  
- Crime percentage by time blocks  

### 3. Trend Analysis
- Yearly crime trend line chart  
- Monthly comparison across years  

### 4. Comparative Analysis
- Arrest vs No Arrest percentage  
- Domestic vs Non-Domestic cases  
- Severe vs Non-Severe crime distribution  

## Interactivity
- Filters added for:
  - Year  
  - District  
  - Crime Type  
- Enabled dynamic exploration of data  

## Tools Used
Tableau, Data Visualization, Dashboard Design  

## Outcome
- Identified crime hotspots and peak crime periods  
- Enabled interactive exploration of crime trends  
- Provided insights to support better resource planning and decision-making
