# Restaurant Tips Analysis (Excel)

## Overview
This project focuses on analyzing a restaurant tips dataset using Excel to understand factors influencing tipping behavior and prepare data for further statistical analysis.

## Data Preparation
- Cleaned the dataset by removing unnecessary columns  
- Converted categorical variables into numeric values using IF formulas (Sex, Smoker, Day, Time)  
- Ensured all key fields such as Total Bill, Tip, and Size were in numeric format  

## Analysis Performed

### Correlation Analysis
- Used Excel Data Analysis ToolPak to generate a correlation matrix  
- Identified relationships between tip amount and influencing factors  

### Pivot Table & Visualization
- Created pivot tables to summarize tips by gender  
- Built pie charts to visualize distribution of tips  

### Comparative Analysis
- Used SUMIFS to create summary tables  
- Developed bar charts comparing tips by gender and time (Lunch vs Dinner)  

## Tools Used
Excel, Pivot Tables, Data Analysis ToolPak, Charts (Pie & Bar)

## Outcome
- Identified key patterns in tipping behavior  
- Built a structured dataset ready for regression analysis  
- Enabled better understanding of how different factors impact tips
